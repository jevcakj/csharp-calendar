Detailed Description

Class representing a collection of calendar events. Implements the IEvents interface to handle event filtering and retrieval. This class uses the HttpClient to interact with a server and retrieve events based on date bounds.

Template Parameters
TThe type of events in the collection, which must be derived from CalendarEventBasic.
